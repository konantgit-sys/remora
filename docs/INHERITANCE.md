# Remora — Наследие и независимость

Remora — полностью самостоятельный проект, без зависимостей от внешних кодовых баз.
Вдохновлён концепцией «рыбы-прилипалы»: не паразит, а компаньон.

## Реализовано самостоятельно

| Компонент | Описание |
|-----------|----------|
| `remora_adapter.py` | Nostr-адаптер, websocket прямая публикация на 40+ релеев |
| `memory.py` | SQLite БД: диалоги, авторы, blacklist, тона, rate-limiter |
| `scanner.py` | Сканирование релеев, сбор kind:1 постов |
| `evaluator.py` | LLM-оценка A/B/C/D через Mistral API |
| `generator.py` | 3 режима генерации + контент-политика |
| `publisher.py` | Публикация с rate-limit и retry |
| `tracker.py` | Мониторинг диалогов в реальном времени |

## Внешние зависимости

- `nostr_sdk` — Python-биндинги для Nostr (подпись событий)
- `websocket-client` — подключение к релеям
- `requests` — вызов Mistral API
- `Mistral AI API` — LLM для оценки и генерации

Remora не использует код других проектов.
