# Remora 🐟

**Паразитический агент влияния на Nostr.**

Remora — рыба-прилипала. Не пишет в пустоту. Живёт в ответах на чужие посты. Каждое сообщение Remora — это reply к мыслям реальных людей. Она не конкурирует за внимание — она использует чужое.

**Версия:** 1.0 (прототип)
**Статус:** Спецификация утверждена
**Основа:** Наследует инфраструктуру Remora V10 (релеи, LLM-прокси, Nostr-адаптер, тона)

---

### Чем отличается от Remora

| Remora | Remora |
|--------|--------|
| Пишет сам в ленту | Пишет в ответ на других |
| Ждёт, пока увидят | Гарантированно видят (автор оригинала) |
| 264 поста/нед, engagement ~0 | 15-20 ответов/день, цель: 1 реакция на 5 комментов |
| Монолог | Диалог |
| LLM: $30/мес | LLM: ~$1.50/мес |

---

### Принципы

1. **Ни одного поста в пустоту** — Remora никогда не публикует standalone-посты. Только replies.
2. **Не быть спамом** — каждый ответ должен быть ценнее, чем средний комментарий человека.
3. **Память диалогов** — Remora помнит, что сказала. Не повторяется. Продолжает треды.
4. **Самообучение** — на ответы, которые получили реакции — повышать вес. На игнорируемые — понижать.
5. **Реальная экономика** — LLM-бюджет ~$1.5/мес. Никаких дорогих экспериментов.
